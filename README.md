
📊 Customer Behavior Analysis Project

📌 Overview
This project focuses on analyzing customer behavior using a complete data analytics workflow. It includes data loading, cleaning, exploratory data analysis (EDA), SQL-based querying, and building an interactive Power BI dashboard to generate actionable business insights.

📂 Dataset
The dataset contains customer purchase information such as:
Customer ID
Product Category
Purchase Amount
Age Group
Subscription Status
Review Ratings
Source: Sample / Public dataset (can be updated if needed)

🛠️ Tools & Technologies
Python (Pandas, NumPy, Matplotlib, Seaborn)
MySQL (Data querying & analysis)
Power BI (Dashboard creation)
Gamma (Presentation creation)
Jupyter Notebook
🔄 Project Workflow
1. Data Loading
Imported dataset using Pandas
Checked structure, data types, and summary statistics
2. Data Cleaning
Handled missing values
Removed duplicates
Corrected data types
3. Exploratory Data Analysis (EDA)
Analyzed:
Revenue by category
Customer age group trends
Purchase behavior patterns
Visualized data using charts and graphs
4. SQL Analysis
Imported cleaned data into MySQL
Performed queries such as:
Revenue by category
Customer segmentation
Subscription analysis
5. Dashboard (Power BI)
Built an interactive dashboard including:
KPI Cards (Revenue, Ratings, Customers)
Revenue by Category & Age Group
Subscription Distribution
Seasonal Trends
6. Reporting & Presentation
Created a detailed report with insights
Designed a professional PPT using Gamma
📊 Dashboard Highlights
Clothing category generates the highest revenue
Young Adults are the top contributing customers
Subscription rate is relatively low
Seasonal sales remain consistent
📈 Key Results
Identified high-performing categories and customer segments
Provided business recommendations to improve revenue
Highlighted areas needing improvement (low-performing categories & subscription rate)
